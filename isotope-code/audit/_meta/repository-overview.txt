ISOTOPE REPOSITORY OVERVIEW
===========================

APPLICATION:
IsotopeAI is a self-hosted AI study planner designed for JEE/NEET/Board students. It features automated study scheduling, task management, and AI-powered study assistance.

ARCHITECTURE:
- Frontend: Vite/React (served from public/ directory after build).
- Backend: Node.js (server.mjs) using a simple express-like architecture with native modules.
- Database: Supabase (PostgreSQL with RLS).
- PWA: Supports offline mode and local data sync.

KEY SUBSYSTEMS:
- Auth: Integrated with Supabase Auth + local Admin panel for owners.
- Backup/Restore: JSON-based backup system with Supabase Storage integration.
- AI Proxy: Routes for Gemini, Groq, and YepAPI to enable chat and summarization.
- Admin Panel: Diagnostic tools and manual schema patch application via /__admin.

REPOSITORY STRUCTURE:
- /server.mjs: Main entry point.
- /public/: Built frontend and static assets.
- /sql/: Database schema migrations and patches.
- /scripts/: Utility scripts for maintenance and CI.
- /audit/: Autonomous audit documentation (current directory).

VERSION INFO:
- Current Version: 3.3.8
- Model: Open-source self-hosted (Clone -> .env -> isotope start).
